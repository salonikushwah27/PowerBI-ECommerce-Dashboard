# E-Commerce Sales Dashboard 🛒

## Overview
Interactive Power BI dashboard analyzing 34,500 e-commerce orders across multiple regions, categories, and payment methods.

## Key Metrics
- Total Revenue: 5.87M
- Total Orders: 34.5K
- Return Rate: 5.52%
- Avg Profit Margin: 28.12

## Features
- Sales by Region & Category
- Payment Method Analysis
- Revenue Trend by Month
- Returns by Category
- Age vs Profit Margin Analysis

## Tools Used
- Power BI Desktop
- DAX (Return Rate measure)

## Dataset
- 34,500 rows
- Fields: Order ID, Category, Region, Payment Method, Profit Margin, etc.
